# Как выложить прототип «ИИ-шлюз» в интернет (без опыта в программировании)

Инструкция для размещения сайта на **бесплатных** сервисах. Вы получите ссылку, по которой прототип будет открываться с любого телефона или компьютера.

---

## Что понадобится

- Компьютер, на котором уже открыт проект в Cursor (папка `ai-gateway-prototype`).
- Электронная почта или аккаунт Google/GitHub для регистрации на бесплатном сервисе.
- 10–15 минут времени.

---

# Способ 1: Netlify (проще всего — только перетащить папку)

Сайт будет размещён на сервисе **Netlify**. Регистрация и один сайт — бесплатно.

---

## Шаг 1. Подготовить папку для загрузки

1. Откройте в Cursor **встроенный терминал**: меню **Terminal** → **New Terminal** (или сочетание клавиш для терминала).
2. Введите по очереди две команды, после каждой нажимайте **Enter**:

   **Первая команда** (переход в папку проекта):
   ```
   cd /Users/svruchushkin/ai-gateway-prototype
   ```

   **Вторая команда** (создание папки с готовым сайтом):
   ```
   npm run build
   ```

3. Дождитесь окончания (появится надпись вроде «built in … ms»). Ошибок быть не должно.
4. В папке проекта `ai-gateway-prototype` появится папка **`dist`**. Именно её мы будем загружать в интернет.  
   Чтобы её было удобно найти: в Cursor в левой панели (обозреватель файлов) откройте папку `ai-gateway-prototype` и найдите папку **`dist`**.

---

## Шаг 2. Зарегистрироваться на Netlify

1. Откройте в браузере сайт: **https://www.netlify.com**
2. Нажмите **«Sign up»** (Войти / Зарегистрироваться).
3. Выберите вход через **Email**, **Google** или **GitHub** — как вам удобнее. Заполните данные, если просят. Регистрация бесплатная.

---

## Шаг 3. Создать сайт и загрузить папку

1. После входа вы попадёте в личный кабинет Netlify. Нажмите **«Add new site»** (Добавить сайт) → **«Deploy manually»** (Развернуть вручную).
2. На экране появится большая область с подписью вроде **«Drag and drop your site output folder here»** (Перетащите сюда папку с сайтом).
3. На своём компьютере откройте папку **`dist`** (она внутри `ai-gateway-prototype`).  
   Выделите **все файлы и папки внутри `dist`** (не саму папку `dist`, а её содержимое: файл `index.html` и папка `assets`).
4. Перетащите это содержимое в окно браузера в область Netlify (drag and drop).
5. Подождите 1–2 минуты. Netlify загрузит файлы и соберёт сайт.
6. Когда появится надпись **«Site is live»** (Сайт запущен), под ней будет ссылка вида:  
   **https://какое-то-название-123.netlify.app**  
   Это и есть адрес вашего прототипа в интернете.

---

## Шаг 4. Открыть сайт с другого устройства

- Скопируйте эту ссылку и откройте её на телефоне, планшете или другом компьютере — прототип «ИИ-шлюз» будет доступен без настройки.

---

## Если нужно обновить сайт позже

1. Снова в терминале Cursor выполните в папке проекта:
   - `cd /Users/svruchushkin/ai-gateway-prototype`
   - `npm run build`
2. Снова откройте в Netlify ваш сайт → вкладка **«Deploys»** → перетащите в область загрузки **содержимое** обновлённой папки `dist`. Старая ссылка останется той же, изменится только содержимое сайта.

# Способ 2: Vercel (через сайт, без командной строки после первой настройки)

Тоже бесплатный сервис. Удобен, если позже захотите подключать обновления через GitHub.

---

## Шаг 1. Зарегистрироваться на GitHub

1. Откройте **https://github.com**
2. Нажмите **«Sign up»**. Создайте аккаунт (email и пароль). Это бесплатно.

---

## Шаг 2. Подготовить папку для загрузки (без лишних файлов)

На GitHub нельзя загрузить больше 100 файлов за раз. В папке проекта есть служебная папка **node_modules** (тысячи файлов) — её загружать не нужно. Сначала создайте «лёгкую» копию проекта:

1. Откройте в Cursor **терминал** (Terminal → New Terminal).
2. Выполните по очереди:

   ```
   cd /Users/svruchushkin/ai-gateway-prototype
   ```
   Нажмите Enter.

   Затем:
   ```
   bash prepare-upload.sh
   ```
   Нажмите Enter.

3. В папке проекта появится папка **`ai-gateway-for-github`**. В ней только нужные файлы (примерно 50–60 штук), без `node_modules` и без `dist`. Эту папку можно спокойно загружать на GitHub.

---

## Шаг 3. Создать репозиторий и загрузить папку

1. Войдите в GitHub. Нажмите зелёную кнопку **«New»** (или **«Create repository»**).
2. В поле **Repository name** введите, например: **ai-gateway**. Остальное можно не менять. Нажмите **«Create repository»**.
3. На открывшейся странице найдите блок **«uploading an existing file»** и нажмите ссылку **«uploading an existing file»**.
4. На компьютере откройте папку **`ai-gateway-for-github`** (она находится внутри `ai-gateway-prototype`). Выделите **все файлы и папки внутри** этой папки (папку `src`, файлы `index.html`, `package.json` и т.д.) и перетащите их в окно браузера GitHub. Либо нажмите «choose your files» и по очереди выберите папку `src` и все файлы из корня `ai-gateway-for-github`.
5. Внизу страницы нажмите **«Commit changes»**. Дождитесь загрузки.

---

## Шаг 4. Подключить репозиторий к Vercel

1. Откройте **https://vercel.com** и нажмите **«Sign Up»**. Войдите через **GitHub** (так проще всего).
2. После входа нажмите **«Add New…»** → **«Project»**.
3. В списке репозиториев выберите **ai-gateway** (или как вы назвали репозиторий). Нажмите **«Import»**.
4. Настройки можно не менять — нажмите **«Deploy»**.
5. Подождите 1–2 минуты. Появится страница с зелёной галочкой и ссылкой вида **https://ai-gateway-xxx.vercel.app** — это адрес вашего прототипа.

---

## Шаг 5. Открыть сайт с других устройств

Скопируйте ссылку и откройте её на любом устройстве — прототип будет доступен в интернете.

---

## Обновление сайта на Vercel (когда появятся изменения в проекте)

Нужно загрузить обновлённые файлы в тот же репозиторий на GitHub (через страницу репозитория → «uploading an existing file» или через Cursor/Git, если кто-то поможет). После загрузки Vercel обычно сам пересобирает сайт; ссылка остаётся прежней.

---

# Краткая памятка

| Действие | Способ 1 (Netlify) | Способ 2 (Vercel) |
|----------|--------------------|-------------------|
| Регистрация | Netlify (email/Google/GitHub) | GitHub + Vercel |
| Подготовка | В терминале: `cd …` и `npm run build` | Загрузить весь проект на GitHub |
| Размещение | Перетащить содержимое папки `dist` в Netlify | Подключить репозиторий в Vercel и нажать Deploy |
| Результат | Ссылка вида …netlify.app | Ссылка вида …vercel.app |

Оба варианта **бесплатные** и подходят для демонстрации прототипа стейкхолдерам с любого устройства.
